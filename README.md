# Политика конфиденциальности BerkHelperMarket

Дата вступления в силу: 13.06.2026

## 1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки данных пользователей Telegram-бота BerkHelperMarket.

Используя бота, пользователь соглашается с условиями настоящей Политики.

---

## 2. Какие данные обрабатываются

При использовании бота могут обрабатываться:

* Telegram ID пользователя;
* имя и username, предоставляемые Telegram;
* сообщения и запросы, отправленные боту;
* информация о количестве доступных генераций;
* данные о платежах и приобретённых пакетах генераций;
* техническая информация, необходимая для работы сервиса.

---

## 3. Цели обработки данных

Данные используются исключительно для:

* предоставления функционала бота;
* генерации и улучшения карточек товаров;
* ведения учёта генераций;
* обработки платежей;
* технической поддержки и улучшения сервиса.

---

## 4. Использование сторонних сервисов

Для обработки запросов пользователей может использоваться сервис GigaChat.

Для проведения платежей может использоваться сервис ЮMoney.

Передаваемые данные используются только для выполнения соответствующих функций сервиса.

---

## 5. Хранение данных

Данные пользователей могут храниться в базе данных сервиса в течение срока использования бота.

Владелец сервиса принимает разумные меры для защиты данных от несанкционированного доступа.

---

## 6. Передача данных третьим лицам

Данные не продаются и не передаются третьим лицам, за исключением случаев:

* когда это необходимо для работы сервиса;
* когда этого требует законодательство;
* при использовании сервисов GigaChat и ЮMoney для обработки запросов и платежей.

---

## 7. Удаление данных

Пользователь может прекратить использование бота в любой момент.

По запросу пользователя сохранённые данные могут быть удалены, если это не противоречит требованиям законодательства и техническим ограничениям сервиса.

---

## 8. Ограничение ответственности

Владелец сервиса не несёт ответственности за действия сторонних платформ и сервисов, включая Telegram, GigaChat и ЮMoney.

---

## 9. Контакты

По вопросам, связанным с обработкой данных, пользователь может обратиться к владельцу сервиса через Telegram или указанный контактный адрес электронной почты.

---

## 10. Изменение политики

Владелец сервиса вправе изменять настоящую Политику конфиденциальности без предварительного уведомления пользователей.

Актуальная версия всегда доступна по опубликованной ссылке.
